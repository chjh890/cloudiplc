## 产品线一览：四条线路，各有所长

CloudIPLC目前主打四条产品线，面向的用户群体完全不同，选之前搞清楚自己的需求最重要。

### 🇭🇰 香港CMI VPS（最受欢迎）

机房在CeraNetworks（CERA），走香港CMI线路，大陆三网直连优化，延迟普遍控制在30ms以内。

适合人群：建站、个人项目、轻量业务、Linux学习。

有个真实用户评价挺有代表性："Linux新手，用来学习研究非常棒！"（来自酷分享社区用户bigdog）

### 🇯🇵 沪日IPLC-NAT专线（延迟怪）

IPLC是什么？International Private Leased Circuit，国际私有租用线路。上海BGP入口，日本NTT/BBIX出口，延迟固定在26ms左右。

注意是"固定"，不是"平均"。普通线路晚高峰可能飙到一两百毫秒，IPLC专线压根不走公网，这个问题不存在。

适合人群：对延迟极度敏感的场景，比如远程办公、实时应用、对网络抖动零容忍的业务。

需要注意：这是NAT产品，共享IP，仅开放1万以上端口，新手购买前想清楚。

### 🇨🇳 泉州电信CN2 VPS（企业向）

走电信AS4809 CN2精品网，直连NTT、PCCW等国际链路，带宽100Mbps起步，流量给得很足。属于大陆IDC，需要实名认证。

适合人群：外贸网站、跨境电商、需要国内出口快速访问海外的业务场景。

价格偏高（月付约¥900起），但考虑到内地电信CN2的IP单价本身就在300-500元/个/月，这个定价在行业里其实不算离谱。个人玩家没必要，真有商业需求的值得认真考虑。

### 🇺🇸 美国洛杉矶VPS（性价比选择）

有QN和C3两个机房，从¥48/月起。C3机房三网直连、电信走163线路，联通用户体验最好。预算有限、主要用来跑点个人项目的，洛杉矶机房性价比不错。
